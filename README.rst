Flasko: A Flask Boilerplate
===========================

(Boostraped from `pybo`_ 0.1)

.. _pybo: https://github.com/fitiavana07/pybo

Installation
------------

Virtualenv::

    make venv

Install `pre-commit` hooks::

    make hooks

Start Development
-----------------

Install locally::

    make install
